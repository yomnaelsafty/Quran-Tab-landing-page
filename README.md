# Qurantab Landing Page

A modern, interactive landing page built using **HTML, CSS, and JavaScript**.  
Focuses on accessibility, smooth animations, and responsive design.

---

## 🚀 Features

- ✅ Keyboard accessibility: Press **Enter** on labels to trigger clicks  
- ✅ Smooth animations when sections scroll into view  
- ✅ Responsive design for mobile and desktop  
- ✅ Clean and modular code structure  

---

## ▶️ How to Use

1. Open `index.html` in your browser  
2. Scroll through the sections to see animations  
3. Click on interactive labels to navigate or trigger actions
